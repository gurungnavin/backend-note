## 1. Initializing an npm Package

### What is npm?
- **npm** = Node Package Manager.
- Used to manage dependencies and scripts for Node.js projects.

### Steps to Initialize
1. Open terminal in your project folder.
2. Run:
   ```bash
   npm init
   ```
- Answer prompts to create ```package.json```.
